## Getting Started:

### Run on local machine
 * Docker: https://www.docker.com/products/docker-desktop
```
docker compose up
```

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
1. Install NPM packages
   ```sh
   npm install --force
   ```
### Usage
```sh
npm start
```
